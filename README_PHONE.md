# Mpitendry Ankadindratombo — fomba mora (amin'ny finday ihany)

## Safidy mora indrindra: GitHub Actions (asaina)
1) Mamoròna kaonty GitHub (raha mbola tsy manana)
2) Mamoròna repository vaovao (public na private)
3) Ampidiro (upload) ao amin'ilay repo ny rakitra rehetra ato amin'ity tetikasa ity
4) Mandehana ao amin'ny Actions > "Build APK" > Run workflow
5) Rehefa vita dia alao (download) ny artefact "mpitendry-apk" (misy ny .apk ao anatiny)

## Fametrahana (installation)
- Alefaso ny “Install unknown apps” ho an'ilay navigateur na file manager ampiasainao
- Sokafy ny .apk dia apetraho
